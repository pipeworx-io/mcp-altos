## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "altos": {
      "url": "https://gateway.pipeworx.io/altos/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use altos
```
